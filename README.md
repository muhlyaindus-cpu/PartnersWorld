# 🌍 Partners World

**Международное сообщество для экспортёров и местных партнёров**

Partners World — это платформа, где экспортёры находят местных экспертов (юристов, логистов, маркетологов) для выхода на новые рынки, а партнёры монетизируют свои знания и связи.

[🐛 Сообщить об ошибке](https://github.com/muhlyaindus-cpu/PartnersWorld/issues)  

## ✨ Возможности

- ✅ **Регистрация и аутентификация** — безопасный вход через JWT
- ✅ **Профили пользователей** — роли экспортёр/партнёр, редактирование профиля, публичные страницы
- ✅ **Запросы на партнёрство** — создание запросов с типом `need` или `offer`, фильтрация по стране
- ✅ **Отклики и чат** — партнёры могут откликаться на запросы (базовая реализация)
- ✅ **Поиск и фильтры** — быстрый поиск по стране и категории услуг
- ✅ **Рейтинги и отзывы** (в разработке)
- ✅ **PostgreSQL** — надёжная база данных
- ✅ **Полностью готов к деплою** — конфигурация через переменные окружения

## 🛠️ Технологии

- **Backend:** Go 1.25+, PostgreSQL, JWT, bcrypt
- **Frontend:** HTML, CSS, JavaScript (vanilla)
- **Инфраструктура:** Docker, GitHub Actions (CI/CD — в планах)

## 📦 Быстрый старт

### Предварительные требования
- Go 1.25 или выше
- PostgreSQL (локально или через Docker)
- Git

### Установка и запуск

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/muhlyaindus-cpu/PartnersWorld.git
   cd PartnersWorld

Установи зависимости:

bash
go mod download
Настрой базу данных:

Убедись, что PostgreSQL запущен

Создай базу partners_db

Выполни миграции из папки migrations/ (или используй встроенные в db.RunMigrations())

Запусти сервер:

bash
go run cmd/api/main.go
Открой браузер: http://localhost:8080

Использование Docker
bash
cd docker
docker compose up -d
PostgreSQL будет доступен на порту 5432, pgAdmin — на 5050 (логин: admin@partnersworld.com, пароль: admin123).
